A web application to help you organize chores, or tasks between you and your friends.

About Page

James Le, David Howes, Ann Ho, Dan Couturier

ChOrganize is a free website designed to be used for groups to organize their chores using a simple and enjoyable interface. This website would be used, for example, as a way for a group of apartment friends to delegate their weekly chores. Our website will allow users to own and be a part of multiple groups at once, each with their own chores and members. Within each of these groups, users are able to view all current chores as well as the current status of that chore, ongoing or complete. In addition, any user which is a member of that group has the ability to add new members and chores. Furthermore, chores can be clicked on in order to view more detailed information about the task. Such information includes chore name, chore description, chore due date, assignee, and group name. This creates a dynamic system in which users are able to interact with one another by creating, completing, and deleting chores. To encourage users to complete chores, and so that group members can monitor one another, statistics are given about the total number of chores completed against the chores completed by each user. Finally, there is a page called My Chores, which allows the user to view all chores that are currently assigned to them regardless of what group the chores belong to. This allows the user to receive a summary of all of their relevant data in a simple and concise manner, avoiding the arduous process of clicking through every single group.
